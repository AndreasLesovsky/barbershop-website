# Barbershop-Website

Eine Website für einen Barbershop, erstellt mit HTML, CSS, JavaScript und PHP.

## Features
- Übersicht über Dienstleistungen und Preise
- Kontaktformular
- Responsives Design mit abwechselnd hellen und dunklen Sektionen
- Terminreservierung mit E-Mail Bestätigung

## Installation
1. Repository klonen: `git clone https://github.com/username/barbershop-website.git`
2. Die Datebank db_friseur muss erstellt werden.
3. `index.php` im Browser öffnen.
